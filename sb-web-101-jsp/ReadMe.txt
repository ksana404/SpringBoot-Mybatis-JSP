================================================ web + mybatis +jsp
----
	功能：做一个简单的 能访问数据库 使用jsp 展示页面 的项目 
	1.把项目下载下来,解压导入 支持 springboot IDE,-->导入项目类型是 maven 项目 
	2.从sql文件中拿到 jtdb.sql 导入数据库
	3.把项目 的数据库名字 和密码 修改成 本地数据库.
	4.springboot App 启动
	5. 浏览器访问: 
	http://localhost:8090/user/findJSP
	
	6.出现人员列表 说明项目运行成功!!!
